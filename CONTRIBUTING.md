# Contributing

Contributions are welcome for:

- corrections to mathematical derivations
- exact reproduction links for cited papers
- additional MMALS baselines and ablations
- source-aware Bayesian MTP models
- verifier calibration experiments
- translations

Please distinguish clearly between:

1. published results,
2. educational reconstructions,
3. MMALS/STRAT-Q hypotheses,
4. empirical results from new experiments.
